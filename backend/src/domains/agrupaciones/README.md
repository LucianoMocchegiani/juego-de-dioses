# Dominio Agrupaciones

DTOs y rutas de **agrupaciones**. Endpoints: `GET /api/v1/bloques/{id}/agrupaciones`, `GET /api/v1/bloques/{id}/agrupaciones/{aid}`. Usa `ParticleResponse` desde `domains/particles/schemas` para `AgrupacionWithParticles.particulas`.

Schemas: `AgrupacionBase`, `AgrupacionResponse`, `AgrupacionWithParticles`.
