# chore(database): Eliminar código obsoleto y archivos deprecados

## Resumen

Elimina archivos deprecados, scripts temporales y carpetas vacías del módulo `database` del backend. Actualiza la documentación para reflejar la estructura real del proyecto.

## Motivación

Existen archivos y carpetas obsoletas que generan confusión y aumentan la superficie de mantenimiento. El archivo `tree_templates.py` está marcado como DEPRECADO y fue reemplazado por el sistema modular `templates/trees/`. Los scripts temporales de migración y debugging ya no se usan. Las carpetas vacías preparadas para funcionalidades futuras no implementadas pueden eliminarse y recrearse cuando se necesiten.
